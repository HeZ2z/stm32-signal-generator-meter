# Roadmap

## 当前状态

- 当前主线阶段：`M7 完成`
- 当前后续阶段：`M8-M11 已规划`

说明：

- 主固件已经集成 `UART + PWM + TIM PWM Input + 串口命令 + LED`
- `PWM` 输出、串口参数调整和测量结果换算已经可构建、可测试
- `PB6 -> PA7` 实板回环已经验证通过，默认测量和断线退化行为符合预期
- `M5` 的误差展示、演示脚本和输出格式收尾已完成
- 当前 `ALIENTEK 4342 RGBLCD`、`GT9xxx` 触摸和 `PB6 -> PA7` 回环主线已联调通过
- 当前已完成 `UART + PWM + TIM PWM Input + LCD + 触摸` 闭环
- 当前后续工作转为真实波形显示、双通道模拟输出与 `XY/YT` 图形演示

## M7 触控调参集成

目标：

- 接入 `GT9xxx` 电容触摸
- 用 LCD 触摸按键直接调节 `PWM` 频率和占空比
- 保留 `UART` 可写诊断与调参能力

退出条件：

- 上电后可直接进入 LCD 状态页
- 触摸按键能稳定触发参数调整
- 通过触摸把参数调整到 `2000/30`，串口和 LCD 同步显示对应 `MEAS`
- 回环断开和重新接回场景都可解释、可演示
- 推荐主演示范围固定为 `1000/50`、`2000/30`、`5000/70`
- 当前状态：已完成主线联调，后续只做文档和提交收尾

## M8 真实波形图显示

目标：

- 把 LCD 从文本状态页推进到真实采样 `YT` 波形页
- 波形图数据必须来自真实采样，不使用软件预览
- 先只稳定显示单通道波形

固定方案：

- 输出源固定复用当前 `PB6(TIM4_CH1)` PWM
- 回采固定为 `PB6 -> PA0(ADC1_IN0)`
- 采用 `ADC1 + DMA` 连续采样单通道
- 保留 `UART` 作为采样状态和异常提示兜底

完成结果：

- LCD 可稳定显示单通道真实方波
- 改变 PWM 参数后，LCD 波形同步变化
- 断线或无效输入时，LCD 明确显示采样无效状态
- 顶部 `INPUT / OUTPUT` 卡与当前波形页语义一致
- 波形刷新频闪已压制到可接受范围

当前实现：

- `signal_capture_adc` 已接入 `ADC1 + DMA` 单通道连续采样
- `scope_render` 已接入单通道 `YT` 波形绘制
- LCD 当前页面已从旧状态页推进到真实采样波形页
- 左侧 `INPUT` 卡固定表示 `PA0 ADC1` 真实输入，右侧 `OUTPUT` 卡显示当前 `PB6 TIM4` 输出参数
- 方波输入在可稳定估算窗口内显示真实输入 `F/D`
- 超出占空比稳定估算窗口时，左卡显示 `F=<当前输出频率> D=--`
- 断线/悬空输入通过 `PA0` 下拉和更严格的方波判定退化为无效输入
- 当前状态：已完成，后续只做文档和提交收尾

## M9 双 DAC 双方波

目标：

- 把新增演示主线切到双路 `DAC`
- 两路通道先只支持方波
- LCD `YT` 扩展为双通道叠加显示

固定方案：

- 输出固定为 `PA4(DAC1)`、`PA5(DAC2)`
- 回采固定为 `PA4 -> PA0`、`PA5 -> PA6`
- `DAC` 实际实现为 `TIM6 + 单 DMA packed dual write(DHR12RD)`
- `ADC` 实际实现为 `TIM2 TRGO + ADC1 双 rank 扫描 + DMA`
- 当前旧 `PWM + Input Capture` 路径保留为可编译、可回归能力，但默认不初始化

退出条件：

- 双路方波输出稳定
- 双通道回采稳定
- LCD `YT` 可同步叠加显示两路波形
- 默认主页面、串口状态和触摸控制都不再依赖旧 `PWM`

## M10 三角波 + 最小 XY

目标：

- 在双通道框架内加入三角波
- 最小化接入 `XY` 模式，不拖到最后才首次落地

固定方案：

- 当前开放波形种类：`square`、`triangle`
- `YT` 继续保留双通道叠加
- `XY` 先做最小版：边框、中心线、逐点绘制
- 当前不做持久辉光、抗锯齿、双缓冲特效

退出条件：

- 方波与三角波都能正确输出和显示
- `XY` 模式可稳定成图
- 不同频率比和相位差下轨迹有可解释变化

## M11 正弦波 + 李萨如收口

目标：

- 完成三种波形：`方波 / 三角波 / 正弦波`
- 完成 `YT / XY` 双模式
- 完成李萨如答辩演示链路

固定方案：

- 正弦波采用查表输出
- `XY` 固定使用真实双通道采样值映射
- 默认主演示覆盖：同频同相、同频 `90°` 相移、`1:2` 频率比

退出条件：

- 三种波形都能稳定输出与显示
- `XY` 可稳定演示直线、圆/椭圆和 `1:2` 李萨如图
- 文档、演示脚本与验证记录全部收口
