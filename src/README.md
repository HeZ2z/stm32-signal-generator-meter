# Source Layout

本目录用于存放项目源码实现。

当前源码已按功能模块拆分：

- `app/`：应用调度和主循环协调
- `signal_gen/`：PWM 输出与参数换算
- `signal_measure/`：输入测量与结果换算
- `ui/`：串口命令解析与控制
- `display/`：显示输出实现，当前为 UART
- `board/`：板级中断与 MSP 配置
- `system/`：启动、系统文件与运行时支撑

实现顺序建议：

1. 先建立最小启动与 `UART` 输出
2. 再补 `PWM` 输出
3. 最后再做参数调整和展示优化

当前结构为后续 `RGBLCD/LTDC` 集成预留了显示扩展位置，新增屏幕实现时优先放入 `display/`，而不是继续回到平铺源码布局。
